## Introdução

Estamos muito animados de lançar a nossa mais nova feature: **Grupos de caça de bosses** 🏹

Para os _completionists_, o Bosstiary foi uma grande novidade para o Tibia. Mas, assim como em vários outros aspectos do jogo, jogadores solo acabam ficando na desvantagem. Você não chegará muito longe se estiver tentando completar todos os bosses checando cada spawn sozinho, mesmo com a ajuda do nosso <RouteLink href="/bosses">Boss Tracker</RouteLink>.

É por isso que nós criamos os <RouteLink href="/bosses/hunting-groups">Grupos de caça de bosses</RouteLink>. Isso irá ajudar jogadores que querem **encontrar um novo círculo de amigos** para cooperarem e caçarem bosses juntos.

## Como o sistema de notificação funciona?

As notificações de bosses serão enviadas imediatamente para cada dispositivo (PC/Mobile) que você registrar em nosso app. Você pode conferir o status de registro e testar se as notificações estão chegando em seu dispositivo através do nosso menu de **Configurações**. Nosso website usa `Web Push Notifications`, então é preciso que o seu dispositivo seja compatível com esse sistema. Usuários de Mac/iOS/Safari talvez não tenham notificações habilitadas por padrão, uma vez que a Apple começou a dar suporte à essa API recentemente.

## Posso desligar notificações para alguns bosses específicos?

Sim, você poderá ter um controle bem granular sobre as notificações que você quer receber. Você pode desabilitar notificações para bosses específicos ou mesmo desligar todas as notificações de uma vez. Você não irá acordar no meio da noite só porque o seu amigo encontrou um Grorlam 🗿

## Como eu posso customizar o meu grupo de caça?

Você pode customizar o nome do grupo, publicar uma descrição pública dele, ou mesmo compartilhar em um mural de mensagens **privado** visível apenas para membros do grupo. O avatar do grupo é gerado aleatoriamente, mas você pode gerar um novo sempre que quiser 🎲

Um grupo pode ser privado. Um grupo privado pode ser encontrado por outros usuários, mas os membros desse grupo estarão ocultos. Essa opção estará disponível apenas para grupos que tiverem pelo menos um membro <RouteLink href="/exevo-pro"><Pro>Exevo Pro</Pro></RouteLink>.

Também existem ferramentas para gerenciar o grupo. Um `Admin` poderá promover membros para `Moderadores`. Moderadores terão os mesmos privilégios que um admin (editar as informações do grupo, kickar membros, aprovar/rejeitar solicitações, etc), com a exceção de alterar a role de outros membros.

Se algum membro começar a sacanear enviando notificações falsas de bosses, é o papel do admin/moderadores kická-lo do grupo 🔨

## Como eu posso deletar um grupo?

Quando todos os membros tiverem saído de um grupo (ou sido kickados), incluindo você, o grupo será desfeito.

## Existem limites para essa feature?

A princípio não. Você pode criar/entrar em quantos grupos você quiser. Notificações também podem ser enviadas de forma irrestrita. É claro, se houver abusos em nossos serviços, é possível que a gente precise lidar com esse vandalismo configurando alguns limites.

Inclusive, seja responsável quanto ao conteúdo postado em nosso site. Nós não pretendemos moderar o conteúdo por aqui, mas certos tipos de conteúdo não serão tolerados (coisas ilegais, por exemplo)

## Posso usar os grupos para outras finalidades?

Sinta-se à vontade para usar como quiser. Se a gente perceber que as pessoas estão usando os grupos para outras finalidades (procurando time para bosses, formando times de caça, etc), nós iremos encarar isso como uma oportunidade para expandir com novas features para esses caso de uso 🚀
