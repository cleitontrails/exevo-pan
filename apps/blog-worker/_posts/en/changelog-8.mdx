## Announcement

Let's kick off 2023 with some great new Bazaar features: **Find similar** and **Favorites**!

Both these new features were designed to help you while searching for a specific type of characters and keep track of their prices 🔍

A lot more has changed since our last <SlugLink href="changelog-7">changelog</SlugLink>, so check out the full list of changes:

## 🎉 Features

- Find similar auctions
- Favorite auctions
- Current applied filters now appears on the top of the auction grid
- Server filters options are now dynamic, reacting to other server filters (as suggested by <ExternalLink href="https://github.com/xandjiji/exevo-pan/issues/162">mateusmsant</ExternalLink>)
- Adding filters and tags for **Primal Ordeal** quest (as suggested by <ExternalLink href="https://github.com/xandjiji/exevo-pan/issues/126">Wedz0ff</ExternalLink>)
- You might find some relevant suggested articles placed around the entire website
- New sexy <ExternalLink href="https://i.imgur.com/ujarjsT.png">preview images thumbnails</ExternalLink> when sharing Exevo Pan links!
- <RouteLink href="/exevo-pro">Exevo Pro</RouteLink> users now have access to more discounts when using the <RouteLink href="/highlight-auction">Auction Highlight</RouteLink> feature
- Text-based search for mounts and outfits filters

## 🔧 Refactors

- The **Falconer Outfit** is now also considered a rare outfit (as suggested by Adriano Montaguti)
- Highlighted auctions will now be highlighted and display their TC investment even with filters applied
- Fixed a bug that would decode <ExternalLink href="https://www.exevopan.com/?auctionId=1061424">nicknames with special characters</ExternalLink> incorrectly
- Nicknames with 4 words or more are now also considered rare
